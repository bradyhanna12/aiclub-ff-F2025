# Fantasy Football Machine Learning Predictor

This project uses machine learning to predict player performance in fantasy football. It aims to help users make data-driven decisions for drafting and managing their fantasy football teams.

## Features
- Predict yearly player scores using historical statistics
- Rank players by projected fantasy points


## Installation

   ```bash
   git clone https://github.com/USERNAME/fantasy-football-ml.git
   cd fantasy-football-ml
2. Create a Virtual Environtment:
